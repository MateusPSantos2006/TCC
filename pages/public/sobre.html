Sobre o Projeto

O sistema de biblioteca digital é uma iniciativa inovadora que tem o potencial de revolucionar a maneira como nos conectamos com a literatura. O projeto resolve o problema da dificuldade de encontrar informações sobre os livros da sala de leitura. Atualmente, os leitores precisam ir até a sala de leitura para saber se um livro está disponível ou não. Com o banco de dados, os leitores poderão pesquisar os livros a partir de palavras-chave, o que tornará o processo muito mais rápido e fácil. Além disso, o projeto permitirá que os professores recomendem livros para suas matérias ou futuros vestibulares. Essa é uma forma de incentivar a leitura e ajudar os alunos a encontrarem livros que sejam relevantes para seus estudos. Esta iniciativa é importante pois, através dela é possível melhorar a vida dos leitores, motivando o aumento da leitura.

O Problema em Questão

A dificuldade de encontrar informações sobre os livros da sala de leitura é um problema que dificulta o acesso aos livros e desencoraja a leitura. Um banco de dados bibliográfico para a sala de leitura resolve esse problema, tornando a pesquisa de livros mais rápida e fácil. Além disso, o banco de dados pode incluir recomendações de livros por parte dos professores, o que ajudará os alunos a encontrarem livros que sejam relevantes para seus estudos. Como objetivos específicos, destacamos: Digitalizar os livros da sala de leitura e armazenar as informações em um banco de dados. Fornecer um sistema de busca e recuperação de informações para facilitar a pesquisa de livros. Tornar a pesquisa de livros mais rápida e fácil incentivando a leitura com o banco de dados bibliográfico. Disponibilizar recomendações de livros dos professores para auxiliar os alunos a encontrarem livros que sejam relevantes para seus estudos. Contribuir com a disseminação da cultura e do conhecimento na comunidade escolar. Para digitalizar as capas dos livros, utilizaremos uma câmera para capturar as imagens ou utilizar uma busca na Internet.
